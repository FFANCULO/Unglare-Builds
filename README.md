# Iron Bloom 176 — Plugin Builds

Pre-built VST3 plugin binaries for **Iron Bloom 176** — spectrally stable analog saturation processor with true stereo, 3D ridgeline spectrum analyzer.

## Downloads

### Windows (x64)
`windows/IronBloom176.vst3` — copy the entire folder to:
```
C:\Program Files\Common Files\VST3\
```

### macOS (coming soon)
`macos/` — built via GitHub Actions. Copy to:
```
~/Library/Audio/Plug-Ins/VST3/
```
Note: unsigned build — right-click → Open on first launch, or run:
```
xattr -d com.apple.quarantine IronBloom176.vst3
```
